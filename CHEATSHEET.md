## 🎯 功能直达

| 激活词 | 功能 | 说明 |
|--------|------|------|
| `军机处` / `看板` | 打开看板 | http://127.0.0.1:7891 |
| `奏折` | 查询已完成任务 | 查看历史任务 |
| `官员` | 查询 Agent 状态 | 查看各 agent 健康状态 |
| `上朝` | 启动服务 | 启动 Tang Political System |
| `退朝` | 停止服务 | 停止 Tang Political System |

---

## 🔄 工作流程

```
用户消息 → main 检测激活词 → 转发给对应 agent → 执行 → 返回结果
```

### 示例流程

1. **复杂任务**："朕要设计一个用户系统"
   - main → zhongshu(中书省) → menxia(门下省) → shangshu(尚书省) → 六部执行

2. **单一任务**："宣兵部，修复登录bug"
   - main → bingbu(兵部) → 直接执行

3. **查询状态**："官员"
   - main → 返回各 agent 状态

---

## 📝 非激活词消息

**不包含以上激活词的普通消息**，由 main agent **直接处理**，不转发给 Tang Political System。

---

## 💡 使用建议

1. **复杂任务** → 使用皇帝专属词（朕/圣旨/下旨）
2. **特定领域任务** → 使用部门直调（宣兵部/户部等）
3. **快速查询** → 使用功能直达（奏折/官员/看板）
4. **不确定时** → 直接描述需求，main agent 会判断
